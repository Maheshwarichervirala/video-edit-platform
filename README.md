# 🎬 Video Edit Platform

A full-stack web application that enables users to explore video editors, upload content, and discover AI-powered video tools — built with a modern dark UI.

---

## 🚀 Features

- 🔍 Browse video editors and tools  
- 👤 Create and view editor profiles  
- 📤 Upload and manage video-related content  
- 🤖 Explore AI video tools  
- 🌙 Modern dark-themed UI  

---

## 🛠️ Tech Stack

**Frontend**
- HTML, CSS, JavaScript

**Backend**
- Node.js  
- Express.js  

**Database**
- MongoDB Atlas
